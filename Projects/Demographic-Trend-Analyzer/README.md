Project: Demographic Trend & Historical Data Analyzer

Engineered a scalable Data ETL pipeline in Python to automate the ingestion and cleaning of 100+ historical datasets, processing over 1 million+ records using Pandas.
Developed high-impact data visualizations using Matplotlib to analyze a century of population shifts, translating raw unstructured text files into actionable demographic insights.
Optimized large-scale data manipulation by implementing Pivot Tables and vectorized grouping operations, reducing data processing time for multi-year trend analysis.
Identified socio-economic naming patterns by designing frequency-based ranking algorithms, successfully isolating top-tier naming trends across diverse historical epochs.
