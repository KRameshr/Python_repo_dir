Project : Census Income Predictive Engine (Machine Learning)


Architected a classification pipeline using Python and Scikit-Learn to predict individual income brackets from 48,000+ records, achieving a high-performance 85% accuracy rate.
Implemented an automated ETL & Preprocessing layer to clean unstructured datasets, handling missing values and utilizing Label Encoding to transform 10+ categorical features into machine-readable formats.
Engineered and compared multiple ML algorithms, including Logistic Regression and Random Forest, to identify the most effective model for socio-economic forecasting.
Conducted Feature Importance analysis to isolate the top 10 socio-economic drivers of income, utilizing Matplotlib and Seaborn to visualize complex data correlations.
Evaluated model reliability through deep-dive metrics, including Confusion Matrices, F1-Scores, and Precision-Recall reports, ensuring high predictive confidence across imbalanced datasets.
